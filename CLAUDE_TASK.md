## Виправлено всі проблеми з діалогом дихальних вправ:

### 1. Виправлено таймер - тепер рахує реальні секунди
**Проблема:** Таймер йшов дуже швидко (delay 100ms + elapsed +1 = 10x швидше)
**Рішення:**
- Використовуємо `System.currentTimeMillis()` для точного відліку часу
- `delay(50)` для плавної анімації з правильним розрахунком прогресу
- Файл: `BreathingViewModel.kt` (метод `startBreathing()`)

### 2. Розраховано правильну тривалість кожної вправи
**Проблема:** Вправи мали фіксовану тривалість (60 сек), що не відповідало паттерну дихання
**Рішення:**
- Додано функцію `calculateActualDuration()` для розрахунку повних циклів
- Кожна вправа тепер має точну тривалість на основі свого паттерну:
  - Діафрагмальне: 4+4=8сек → 7 циклів = 56 сек
  - Квадратне: 4+4+4+4=16сек → 3 цикли = 48 сек
  - 4-7-8: 4+7+8=19сек → 3 цикли = 57 сек
  - Спокійне: 3+5=8сек → 6 циклів = 48 сек
  - Енергійне: 2+2=4сек → 7 циклів = 28 сек
  - Глибоке: 6+6=12сек → 5 циклів = 60 сек
  - Розслаблююче: 4+8=12сек → 5 циклів = 60 сек
  - Ритмічне: 3+3+3+3=12сек → 4 цикли = 48 сек
- Файл: `BreathingState.kt`
