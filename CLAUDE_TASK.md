Потрібно закомітити і запушити всі незакомічені зміни.

=== КОМАНДИ ===

1. Перевір статус:
git status

2. Додай всі файли:
git add .

3. Закоміть:
git commit -m "fix: sync uncommitted changes from laptop 1"

4. Запуш:
git push origin main

Якщо буде конфлікт при push — скажи мені, НЕ роби force push!
