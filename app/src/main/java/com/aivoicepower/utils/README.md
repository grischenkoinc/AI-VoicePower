# Utils

Допоміжні утиліти та розширення.

## Структура:

- **audio/** - Робота з аудіо
  - AudioRecorder, AudioPlayer, SpeechRecognizer

- **extensions/** - Kotlin розширення
  - Context, String, Flow extensions

- **formatters/** - Форматування даних
  - Time, Number formatters

- **constants/** - Константи
  - AppConstants, AnalyticsConstants
