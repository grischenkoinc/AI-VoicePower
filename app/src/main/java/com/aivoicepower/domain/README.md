# Domain Layer

Бізнес-логіка застосунку (Domain Layer) - незалежний від фреймворків шар.

## Структура:

- **model/** - Бізнес-моделі
  - **user/** - Моделі користувача
  - **course/** - Моделі курсів та уроків
  - **exercise/** - Моделі вправ
  - **analysis/** - Моделі аналізу голосу
  - **content/** - Контентні моделі

- **repository/** - Інтерфейси репозиторіїв
  - Абстракції для роботи з даними

- **usecase/** - Use Cases
  - Бізнес-логіка застосунку
