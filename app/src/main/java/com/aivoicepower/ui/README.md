# UI Layer (Presentation)

Шар представлення (Presentation Layer) - інтерфейс користувача.

## Структура:

- **navigation/** - Навігація застосунку
  - NavGraph, Routes

- **screens/** - Екрани застосунку
  - **onboarding/** - Онбординг
  - **diagnostic/** - Діагностика
  - **home/** - Головний екран
  - **warmup/** - Розминка
  - **courses/** - Курси та уроки
  - **improvisation/** - Імпровізація
  - **aicoach/** - AI-тренер
  - **progress/** - Прогрес
  - **results/** - Результати аналізу
  - **settings/** - Налаштування

- **components/** - Переспільні UI компоненти
  - **buttons/** - Кнопки
  - **cards/** - Картки
  - **progress/** - Індикатори прогресу
  - **timer/** - Таймери
  - **audio/** - Аудіо компоненти
  - **feedback/** - Фідбек компоненти
  - **common/** - Загальні компоненти

- **theme/** - Тема застосунку
  - Кольори, типографіка, форми
